# DisplayConverter

A Paper plugin to convert Item Frames and Signs into Item and Text Displays.

## Features
- **Item Frame Conversion**: Punch an Item Frame while in mode (`/convertframe`) to convert it.
- **Sign Conversion**: Right-click a Sign while in mode (`/convertsign`) to convert it to floating text.
    - Preserves text, layout, and **dye colors**.
    - Supports Wall Signs and Standing Signs.
- **Persistence**: Usage is tracked in `plugins/ItemFrameConverter/database.db` (SQLite).
- **Revert**: Restore any converted display back to its original Item Frame or Sign form.
    - Checks for valid placement.
    - Preserves all NBT data (Enchants, Lore, Names).
    - **Smart Revert**: Restores original sign placement, rotation, and glowing text.
- **Audio Feedback**: Satisfying sound effects for all major actions.
- **Management UI**: List, Teleport, Preview, and Delete displays via `/managedisplays`.
    - Native Item Tooltips for 1:1 preview.

## Commands
- `/convertframe` (or `/if2id`): Toggle item frame conversion mode.
- `/convertsign`: Toggle sign conversion mode.
- `/managedisplays`: List your owned displays.
- `/adminmanagedisplays`: (Admin) List ALL displays.

## Permissions
- `itemframeconverter.use`: Use the convert commands.
- `itemframeconverter.manage.self`: Manage your own displays.
- `itemframeconverter.manage.admin`: Manage all displays.

## Configuration
The plugin creates a `config.yml` on first run:
```yaml
view-distance: 10.0 # How far displays are visible
sign-text-scale: 0.4 # Text size scaling
```
